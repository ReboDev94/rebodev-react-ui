## Install

Install with npm or yarn

```bash
npm install yaret-ui
or
yarn add yaret-ui
```

## Quick Start

import component from yaret-ui

```js
import { Input, Button } from 'yaret-ui';

function App() {
  return (
    <div>
      <Input/>
      <Button>Click</Button>
    </div>
  );
}
```
